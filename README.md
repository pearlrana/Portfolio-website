# ✨ Pearl's Portfolio

A modern, interactive portfolio website built to showcase my projects, technical skills, and journey as a Computer Science student.

## 🌐 Live Demo

**Website:** https://pearlrana.github.io/Portfolio/

## 🚀 Features

- Modern and responsive design
- Smooth animations and transitions
- Projects showcase
- Skills section
- About Me
- Contact section
- Downloadable resume

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/pearlrana/Portfolio.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
